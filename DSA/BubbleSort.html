Bubble Sort Code (Java)

public class BubbleSort {
    public static void main(String[] args) {
        int[] arr = {5, 1, 4, 2, 8};
        bubbleSort(arr);
        for (int i = 0; i < arr.length; i++) {
            System.out.print(arr[i] + " ");
        }
    }
    static void bubbleSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n-1; i++) {
            for (int j = 0; j < n-i-1; j++) {
                if (arr[j] > arr[j+1]) {
                    int temp = arr[j];
                    arr[j] = arr[j+1];
                    arr[j+1] = temp;
                }
            }
        }
    }
}

Output:

1 2 4 5 8

What is Bubble Sort in DSA?

Bubble Sort is one of the most fundamental sorting algorithms. It works by repeatedly comparing adjacent elements and swapping them if they are in the wrong order. The process continues until the array is completely sorted.

In computer science, understanding Bubble Sort is important as it introduces basic concepts like comparison-based sorting and helps in learning optimization techniques like flagging.

Though not practical for large datasets due to its O(n^2) time complexity, Bubble Sort is used in educational contexts to teach algorithmic thinking. Students learn about time complexity analysis, algorithm optimization, and the differences between various sorting techniques through it.

This algorithm is also helpful in understanding how more complex algorithms like Quick Sort or Merge Sort reduce time complexity by adopting more advanced strategies. It lays the foundation for learning algorithmic design principles.

Time and Space Complexity

Time Complexity: O(n^2) in the average and worst case, and O(n) in the best case (when the list is already sorted).

Space Complexity: O(1), as Bubble Sort only requires a constant amount of extra space.
