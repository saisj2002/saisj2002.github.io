Selection Sort Code (Java)

public class SelectionSort {
    public static void main(String[] args) {
        int[] arr = {5, 1, 4, 2, 8};
        selectionSort(arr);
        for (int i = 0; i < arr.length; i++) {
            System.out.print(arr[i] + " ");
        }
    }

    static void selectionSort(int[] arr) {
        int n = arr.length;
        for (int i = 0; i < n - 1; i++) {
            int minIndex = i;
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            // Swap the found minimum element with the first element
            int temp = arr[minIndex];
            arr[minIndex] = arr[i];
            arr[i] = temp;
        }
    }
}

Output:

1 2 4 5 8

What is Selection Sort in DSA?

Selection Sort is a simple comparison-based sorting algorithm. It works by repeatedly finding the minimum element in the unsorted portion of the array and swapping it with the first unsorted element.

Although it is not efficient on large datasets, it has the advantage of being easy to implement and requiring minimal additional space.

Time and Space Complexity

Time Complexity: O(n^2) in both average and worst case. The best case is also O(n^2) since the algorithm always scans through the entire unsorted portion.

Space Complexity: O(1), as it is an in-place sorting algorithm.
