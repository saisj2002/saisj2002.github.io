DSA Concepts Index

Concept Description Time Complexity (Best) Time Complexity (Average) Time Complexity (Worst)
Array A linear data structure that holds elements of the same type in contiguous memory. O(1) O(n) O(n)
Linked List A linear data structure where elements are stored in nodes, and each node points to the next one. O(1) O(n) O(n)
Stack A linear data structure that follows the Last In, First Out (LIFO) principle. O(1) O(n) O(n)
Queue A linear data structure that follows the First In, First Out (FIFO) principle. O(1) O(n) O(n)
Hash Table A data structure that stores key-value pairs using a hash function to index into an array. O(1) O(1) O(n)
Binary Search Tree (BST) A tree structure where each node has at most two children, and the left child is smaller than the parent while the right child is greater. O(log n) O(log n) O(n)
Heap A special tree-based data structure that satisfies the heap property (min-heap or max-heap). O(1) O(log n) O(log n)
Graph A data structure that consists of nodes (vertices) connected by edges. Depends on the algorithm Depends on the algorithm Depends on the algorithm
Sorting Algorithms Various algorithms to sort data (e.g., Bubble Sort, Merge Sort, Quick Sort, etc.). O(n log n) O(n log n) O(n^2)
Dynamic Programming A technique used to solve complex problems by breaking them into simpler overlapping subproblems. Depends on the problem Depends on the problem Depends on the problem
