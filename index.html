Concept Description Time Complexity (Best) Time Complexity (Average) Time Complexity (Worst)
Array A linear data structure that holds elements of the same type in contiguous memory. O(1) O(n) O(n)
Linked List A linear data structure where elements are stored in nodes. O(1) O(n) O(n)
Stack A data structure following Last In, First Out (LIFO) principle. O(1) O(n) O(n)
Queue A data structure following First In, First Out (FIFO) principle. O(1) O(n) O(n)
Hash Table A structure that stores key-value pairs using a hash function. O(1) O(1) O(n)
Binary Search Tree (BST) A tree where each node has at most two children, left smaller, right larger. O(log n) O(log n) O(n)
Heap A special tree-based data structure that satisfies the heap property. O(1) O(log n) O(log n)
Graph A structure of vertices connected by edges. Depends on algorithm Depends on algorithm Depends on algorithm
Trie A tree-like data structure used for storing a dynamic set of strings, typically used for fast searching. O(1) O(n) O(n)
Binary Heap A complete binary tree that satisfies the heap property. O(1) O(log n) O(log n)
Segment Tree A binary tree used for storing intervals or segments. It allows querying sums or other operations. O(log n) O(log n) O(log n)
AVL Tree A self-balancing binary search tree where the difference in heights of left and right subtrees is at most 1. O(log n) O(log n) O(log n)
B-Tree A self-balancing tree data structure that maintains sorted data and allows search, insert, delete in logarithmic time. O(log n) O(log n) O(log n)
Floyd-Warshall Algorithm An algorithm for finding the shortest paths in a weighted graph with positive or negative edge weights. O(n^3) O(n^3) O(n^3)
Dijkstra's Algorithm An algorithm for finding the shortest paths between nodes in a graph. O(V^2) O(V^2) O(V^2)

Dynamic Programming A method for solving complex problems by breaking them down into simpler subproblems and solving each subproblem only once. Varies Varies Varies
Quick Sort A divide and conquer algorithm that works by partitioning an array and sorting the parts independently. O(n log n) O(n log n) O(n^2)
Merge Sort A divide and conquer algorithm that splits an array into halves and merges them in sorted order. O(n log n) O(n log n) O(n log n)
Insertion Sort A simple sorting algorithm that builds the sorted array one element at a time. O(1) O(n^2) O(n^2)
Selection Sort A simple sorting algorithm that repeatedly selects the smallest element from the unsorted part. O(1) O(n^2) O(n^2)
Bubble Sort A simple sorting algorithm that repeatedly steps through the list and swaps adjacent elements if they are in the wrong order. O(n) O(n^2) O(n^2)
